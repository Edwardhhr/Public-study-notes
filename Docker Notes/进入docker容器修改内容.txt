查看容器id:
docker ps
这里查看的是registry的id而不是ui的

安装vim文本编辑工具
apt-get install vim -y

进入容器：
docker exec -it 容器id sh
这里有sh用sh有bash用bash

修改密码
vi /etc/registry/auth/htpasswd
按 : 键进入命令行模式。
按 i 键在光标前插入文本
按 o 键在光标所在行的下一行开始新行并插入文本
按 a 键在光标后插入文本

编辑文本：在插入模式下，可以直接输入文本。使用方向键或H、J、K、L键（分别对应左、下、上、右）移动光标。

回到命令模式：完成编辑后，按Esc键回到命令模式。
进入命令行模式：
在命令模式下，按 : 键进入命令行模式。

保存并退出：在命令行模式下，输入wq或x命令，然后按Enter键，即可保存对文件的修改并退出vi编辑器。

只保存不退出：在命令行模式下，输入w命令，然后按Enter键，即可保存文件但不退出vi编辑器。
按:键进入命令行模式。

输入q!命令，然后按Enter键，即可不保存对文件的修改并退出vi编辑器。
